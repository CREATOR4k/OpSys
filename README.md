# OpSys

# Прежде чем начать разработку ОС, потребуется несколько инструментов и базовых знаний:

  1) Ассемблер и C — это языки, на которых чаще всего пишут ядра операционных систем.

  2) Эмулятор (Bochs или QEMU) — позволит тестировать вашу ОС без необходимости постоянно загружать её на реальный компьютер.

  3) Компилятор — для перевода вашего кода на язык, понятный процессору (чаще всего используется GCC).

  4) Загрузчик (Bootloader) — это программа, которая загружает ядро вашей операционной системы. Мы будем использовать GRUB, поскольку он широко поддерживается и удобен для начинающих.
